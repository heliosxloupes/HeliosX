# HeliosX

A Next.js website for HeliosX, a company bringing affordable, reliable, and well-designed surgical loupes to the medical community.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Design System

The design system is documented in `design-system.json` and follows the "Daylight-Inspired-HeliosX" aesthetic:
- Minimal, premium, calm design
- White background, black text, gold accent (#F5B544)
- Large typography, generous white space
- Smooth scroll transitions and subtle animations

## Project Structure

- `app/` - Next.js app router pages and layouts
- `components/` - Reusable React components (Header, Footer)
- `design-system.json` - Design system documentation
